\chapter{Installation}
\label{ch:installation}

Der >>GroupMatcher<< ist eine portable Software \mnote{Portable Software muss nicht installiert werden und kann deshalb auch auf externen Datenträgern mitgeführt werden.}. Während sie auf den Betriebssystemen \hl{Linux} und \hl{MacOS} lediglich die Anwendungsdatei ausführen müssen, um das Programm in ihrer Systemsprache zu starten, ist es auf \hl{Microsoft Windows} zu empfehlen, eine der im Programmverzeichnis enthaltenen Verknüpfungen zu verwenden. Diese sind mit Landeskennungen \mnote{z.B.: >>DE<< oder >>EN<< } versehen, welche sich auf die Sprache der Benutzerfläche beziehen. Die Verknüpfung >>GroupMatcherDE<< wird das Programm in der Ausgangssprache >>Deutsch<< starten.
